# Kabuto - 配布用リポジトリ

J-Quants データを使った中長期投資分析プラットフォーム **Kabuto** の配布用リポジトリです。

## 最新版のダウンロード

最新版は以下からダウンロードできます。

- [macOS Universal DMG](https://github.com/kabuto-jp/kabuto/releases/latest/download/Kabuto-macos-universal.dmg)
- [Windows x64 Installer](https://github.com/kabuto-jp/kabuto/releases/latest/download/Kabuto-windows-x64-setup.exe)
- [Releases ページ](https://github.com/kabuto-jp/kabuto/releases/latest)

macOS版は Apple Silicon / Intel Mac の両方に対応する Universal ビルドです。

## インストール方法

### macOS

1. `Kabuto-macos-universal.dmg` をダウンロードします。
2. DMGを開きます。
3. `Kabuto.app` を `Applications` に移動します。
4. `Applications` からKabutoを起動します。

### Windows

1. `Kabuto-windows-x64-setup.exe` をダウンロードします。
2. インストーラーを実行します。
3. 画面の案内に沿ってインストールします。

## 自動更新

v0.1.1 以降のバージョンには自動更新機能が含まれます。
アプリ起動時に新バージョンが見つかると、画面上部に「Kabuto vX.Y.Z が利用可能です」のバナーが表示されます。

## サポート OS

- macOS 12 以降
- Windows 10 / 11 (x64)
