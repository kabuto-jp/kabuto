# Kabuto - 配布用リポジトリ

J-Quants データを使った中長期投資分析プラットフォーム **Kabuto** の配布用リポジトリです。

## 最新版のダウンロード

[Releases ページ](https://github.com/kabuto-jp/kabuto/releases/latest) から `Kabuto_*_x64-setup.exe` を取得してください。

## 自動更新

v0.1.1 以降のバージョンには自動更新機能が含まれます。
アプリ起動時に新バージョンが見つかると、画面上部に「**Kabuto vX.Y.Z が利用可能です**」のバナーが表示されます。

## サポート OS

- Windows 10 / 11 (x64)

## 開発

ソースコードは別リポジトリで管理されています。バグ報告・機能要望は本リポジトリの Issues までお願いします。
